LifeGuard

О проекте Участники Журнал Ресурсы

Проект по предмету "Проектная деятельность" - LifeGuard

Разрабатывается кроссплатформенное приложение (мобильное, десктопное и веб-версия) для автоматизации ключевых процессов пожарно-спасательных служб;

Стоит задача объединить все необходимые для работы пожарной охраны функции в единой информационной системе;

Проект сочетает практическую значимость для пожарных служб и образовательную ценность для студентов-разработчиков.
